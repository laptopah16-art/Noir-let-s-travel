# Noir Travel ✈️

A modern luxury travel planning platform built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Responsive design
- Animated hero section
- Travel planner
- Destination showcase
- Weather page
- Testimonials and FAQ sections
- GSAP scroll animations
- Framer Motion transitions

## Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- GSAP

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/noir-travel.git
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Open:

```
http://localhost:3000
```

Build for production:

```bash
npm run build
```

Start production server:

```bash
npm run start
```
